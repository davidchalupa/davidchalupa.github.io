GraphCombEx: Graph Combinatorial Explorer

GraphCombEx: Graph Combinatorial Explorer is a light-weight, cross-platform software tool for exploring combinatorial optimization problems in complex networks. It supports a wide range of different functionalities and scalable solutions for graph optimization problems.

Features

Apart from other functionalities, GraphCombEx supports handling of graphs with millions of edges. It currently supports undirected graphs and the following graph metrics: the number of triangles, components, diameter of the main component, maximum clique interval, independent set size interval or dominating set size interval. It also supports several graph visualization methods.
